# Stock Market Data Platform

## Overview
A modern data engineering project that ingests, processes, transforms, and visualizes stock market data using Airflow, Spark, Snowflake, dbt, and Metabase.

## Tech Stack
- Airflow
- Spark
- Snowflake
- dbt
- MinIO
- Docker
- Metabase
